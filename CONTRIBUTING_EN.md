## Project Overview (Current State)

The platform aims to provide an interactive environment to practice **React / Next.js** progressively, starting with fundamentals and evolving to more complex challenges.

Currently:

* The project is functional, but **rough**
* The interface is still provisional
* Not all exercises have been tested to the limit
* Requirements are still being defined

This is expected at this stage.

## How to Contribute

1. Clone the repository
2. Install dependencies:

   ```bash
   npm install
   ```
3. Run the project locally:

   ```bash
   npm run dev
   ```
4. Explore, test, break, experiment

### Creating Contributions

* Create a branch from main
* Branch naming pattern:

  * `feature/feature-name`
  * `fix/fix-name`
  * `refactor/refactor-name`

5. Make your commits
6. Push the branch
7. Open a Pull Request

The code will be reviewed, feedback will be given, and if everything is ok, it will be merged into the main branch.
